# M3-001 — ROI Calculator

## Goal
Pagina `/calculator/roi` — calculator interactiv care arată cât economisește un centru educațional prin trecerea la Vector Learn. Input: dimensiune centru, sistem actual, comportamente. Output: bani salvați, ore recuperate, breakdown pe modul, payback period.

## Acceptance criteria
- [ ] Pagina la `/calculator/roi`
- [ ] 6-8 input-uri (elevi activi, profesori, tool-uri actuale, ore admin/săptămână, rată no-show, restanțe %)
- [ ] Output în timp real: total savings €/lună, ore recuperate, payback period, ROI 12 luni
- [ ] Breakdown pe categorii (timp admin, reducere no-show, plăți la timp, reducere churn, time-to-market campanii)
- [ ] Comparație vizuală "actual" vs "cu Vector Learn"
- [ ] Buton "Trimite raport pe email" (capture lead)
- [ ] CTA "Vezi prețuri"
- [ ] Responsive + dark mode
