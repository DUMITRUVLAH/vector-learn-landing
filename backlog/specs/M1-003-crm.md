# M1-003 — CRM și vânzări

## Goal
Pagina `/modules/crm` cu kanban interactiv (drag & drop între coloane: Lead Nou → Trial → Plătit → Pierdut) și calculator de conversie.

## User stories
- **Manager**: "Văd vizual unde se blochează leadurile?"
- **Vânzător**: "Cum mut un lead între stadii?"
- **Marketing**: "Văd atribuirea UTM și conversia per sursă?"

## Acceptance criteria
- [ ] Pagina la `/modules/crm`
- [ ] Kanban cu 4 coloane, 8 leaduri inițiale, drag & drop între coloane
- [ ] Counter live per coloană
- [ ] Calculator conversie: input "leaduri/lună", "rată trial → plătit (%)", output "elevi noi/lună"
- [ ] Pie chart simplu pentru distribuția pe surse (Facebook, Google, Site, Recomandare)
- [ ] 4 secțiuni: *Pipeline vizual*, *Automatizări*, *Atribuire UTM*, *Telefonie integrată*
- [ ] FAQ 4 întrebări
- [ ] Responsive + dark mode
