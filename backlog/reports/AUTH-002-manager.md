# AUTH-002 — Manager Persona Review (Andreea Mitran)

**Verdict: BUY**

**What I like:**
- Invitation flow is exactly what I need to onboard new staff. No need to share passwords or use "admin creates account then tells them the password" workaround.
- Role pre-selection on invite means the new teacher can't accidentally get admin access.
- 7-day expiry is realistic — new staff often take a few days to accept.
- Auto-login after accepting the invitation is great UX — no "now go log in" step.
- Email pre-filled and read-only on the acceptance page prevents typos.

**Friction:**
- I'd love a list in /app/team showing pending invitations so I can see who hasn't accepted yet and resend if needed. The spec mentions "Retrimite invitație" button but I don't see the team page UI built yet.
- No email verification flow visible in the UI for admins. The banner is for the invited user — fine.

**Blocking issues:** None
