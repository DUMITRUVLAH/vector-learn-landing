# SET-801 — Persona Manager Report (Andreea Mitran)

**Verdict: BUY**

## What was built
- Team management page `/app/settings/team` — invite colleagues by email with role, deactivate departed staff
- Role matrix (owner/admin/manager/teacher/receptionist) — clear permissions per module
- Branding settings — upload logo, set primary color, preview live
- Audit log settings — aggregate view with filters + CSV export
- Notification preferences — per-user opt-in/out per category

## Feedback

LOVES:
- Invite coleg cu rol specific → nu mai dau parola mea de admin directorilor de filială
- Dezactivare instant la plecare angajat → securitate reală
- Matricea de permisiuni vizibilă → pot explica noilor angajați la ce au acces

USEFUL:
- Branding cu logo → demonstrațiile pentru clienți arată profesionist
- Audit log cu filtre → pot investiga incidentele fără să sun IT

WOULD LIKE:
- Bulk invite (import CSV cu colegi) → înscrierea echipei de 15 persoane durează mult
- Notificări push pe telefon → acum vedem alertele doar în browser

## Business case
Pentru o academie cu 6 locații și 20+ angajați, management-ul echipei este blocat
fără această funcționalitate. ROI imediat: 30 min/săptămână economizit per manager
pe gestionarea acceselor.
