# DOCMERGE-002 — Manager Persona Review (Andreea Mitran)

**Verdict: BUY**

The Excel import wizard is a direct productivity tool for Andreea's workflow: she has 80 contracts/month, each with a different student name, sum, and date. The 3-step flow (upload → auto-map columns → preview rows) matches her mental model exactly.

**Strengths:**
- Drag-and-drop Excel file with clear feedback (file name + size shown after drop)
- Auto-map is diacritics-insensitive — "Sumă" maps to `{{suma}}` without manual intervention
- Row preview table (50 visible, total count shown) gives confidence before committing to generation
- Stepper shows progress clearly; "Înapoi" works at every step
- "Generează PDF-uri (în curând)" is honest — doesn't show a button that does nothing silently
- Correct business chrome (BusinessShell, Document Merge section in sidebar)

**Friction points:**
- File dropzone on mobile: works technically but drag-and-drop is hard on touch; the tap-to-select is the real UX on mobile (acceptable)
- Warning about unmapped placeholders uses amber color — correct intent, slightly small text
