# COURSE-203 — Persona Manager Report (Andreea Mitran)

**Verdict: BUY**

## What I reviewed
Promo codes with percent or fixed discount, expiry date, max uses, usage tracking.

## What works
- Campaign codes (BACK2SCHOOL, SUMMER20) are exactly how I run seasonal promos
- Max uses limit: I can share a code on Instagram without fear someone uses it 500 times
- Expiry: automatically invalid after September 15 — no need to remember to disable it
- Usage count: I can tell my owner "codul a fost folosit de 23 ori" without building a report
- Validator widget: staff can test a code before telling a client it's valid

## Quote
"Dau codul TOAMNA20 mamei pe WhatsApp, ea îl dă la recepție, recepția îl introduce și apare automat reducerea de 20%. Fără calcule manuale, fără erori."

## Friction
- Would like to see which payments used each code (a payments filter by promo_code_id)
- Future: ability to disable a code mid-campaign without having to set expiry retroactively

## Verdict explanation
Promo codes are a real sales tool. Competitors (Mindbody, Booksy) all have them. This puts us at feature parity.
