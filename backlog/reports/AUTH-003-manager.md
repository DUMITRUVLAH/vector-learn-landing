# AUTH-003 — Manager Persona Review (Andreea Mitran)

**Verdict: BUY**

**What I like:**
- Profile page has all the fields I need: name, phone, timezone. When staff works from different cities the timezone field is important.
- Language preference (ro/en/ru) is great — we have Russian-speaking teachers.
- GDPR data export button is exactly what auditors ask for. One click and they get a JSON.
- "Ștergere cont" requires password confirmation — prevents accidental deletes.
- 30-day soft-delete grace period is legally correct and gives me time to recover if a teacher deletes their account by mistake.

**Friction:**
- The GDPR section says "anonimizat complet în 30 de zile" but there's no countdown showing the user when the final deletion happens. A note like "Contul tău va fi șters definitiv pe 2 iulie 2026" would be clearer.
- Change password logs out on all devices — that's correct behavior, but the UI should warn users BEFORE they click (not just after).

**Blocking issues:** None
